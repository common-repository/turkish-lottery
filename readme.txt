== Description ==

This plugin shows results of lottery in Turkey by getting data from the website link:https://www.thelotter.com/lottery-results/
 
== Installation ==
1.Download the plugin file 'Lottery'
2.After downloading plugin file upload `Lottery.php` to the `/wp-content/plugins/` directory
3.Activate the plugin through the 'Plugins' menu in WordPress
